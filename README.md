# Sociolobas

> *Belajar IPS bukan hanya menghafal fakta, tetapi memahami kehidupan sosial secara adil dan berkesadaran.*

**Sociolobas** adalah platform pembelajaran IPS berbasis web untuk **SMP (khususnya kelas 7)** yang dirancang dengan pendekatan **kontekstual, reflektif, dan kritis-ramah siswa**. Proyek ini berangkat dari keyakinan bahwa IPS seharusnya dekat dengan realitas hidup peserta didik, tanpa kehilangan kedalaman berpikir.

Sociolobas membantu siswa **belajar membaca lingkungan sosialnya**, bukan sekadar mengerjakan soal.
